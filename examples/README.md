# Examples

This directory contains historical development workflows and local-data examples. These scripts often require FITS, NetCDF, JP2, CSV, or other observation data that is not stored in the repository.

Use `configs/paths.example.yaml` as a starting point for local path configuration. Examples are not collected by the lightweight pytest suite.

## Groups

- `aia_hmi/`: solar-limb and AIA/HMI related examples.
- `radio/`: CSO spectra and FITS metadata examples.
- `radio_aia_hmi/`: AIA, radio source, and HMI overlay development workflows.
