# Maintained public files

- `solar_toolkit/`: installable public package.
- `tests/`: public computation, architecture and packaging contracts.
- `examples/public_api/`: import-only examples using explicit inputs.
- `pyproject.toml`: distribution metadata for `solar-physics-toolkit` 0.3.0.
- `CITATION.cff`: public citation metadata.
- `LICENSE`: license for this Python partition.

The repository-level partition map and publication boundary are documented in
`../../README.md`. Local application files are intentionally absent from this
directory.
